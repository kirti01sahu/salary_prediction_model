# Employee Salary Prediction System using Multiple Linear Regression

## Overview
The Employee Salary Prediction System is a Machine Learning project developed to predict employee salaries based on Age, Gender, Education Level, Job Title, and Years of Experience using Multiple Linear Regression.

## Objective
To build a machine learning model that accurately predicts employee salaries and demonstrates the complete ML workflow.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset Features
- Age
- Gender
- Education Level
- Job Title
- Years of Experience
- Salary (Target)

## Workflow
1. Data Collection
2. Data Cleaning
3. Missing Value Handling
4. Feature Engineering
5. Label Encoding
6. Train-Test Split
7. Multiple Linear Regression
8. Model Evaluation
9. Salary Prediction

## Data Preprocessing
- Mean imputation for numerical columns.
- Mode imputation for categorical columns.
- Label Encoding for categorical features.

## Machine Learning Model
Algorithm: Multiple Linear Regression

### Input Features
- Age
- Gender
- Education Level
- Job Title
- Years of Experience

### Target Variable
- Salary

## Model Evaluation
Metrics Used:
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

Achieved R² Score: ~0.885
